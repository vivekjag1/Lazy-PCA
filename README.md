## Getting started 

## Required packages 
- Pandas 
- Numpy 

## Embeddings generations 
To generate embeddings, simply run the ```embeddings_generation.py``` python script in this repository. 

## Model Comparison 
To compare Lazy PCA, Scikit Learn, and the baseline implementation, simply run the ```run_results.py``` scripts, which runs a comparison of SciKit-Learn, the Power Iteration, and Lazy PCA with varying degrees of orthogonality relaxation.
